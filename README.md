# Pong Classic 1972

A faithful reconstruction of the original 1972 Atari Pong, built for modern browsers.

## Features
- **Authentic 1972 Physics**: Ball speed increases by 5% with every paddle hit.
- **Retro Aesthetics**: Solid black and white graphics with a CRT scanline effect.
- **Classic Scoring**: First to 11 points wins.
- **Audio Feedback**: Procedural square-wave beeps and boops using Web Audio API.

## Controls
- **Player 1 (Left)**: `W` (Up), `S` (Down)
- **Player 2 (Right)**: `UP ARROW` (Up), `DOWN ARROW` (Down)

## Technical Details
- **Resolution**: 640x480 (4:3 Aspect Ratio)
- **Language**: Vanilla JavaScript (HTML5 Canvas)
- **Font**: Press Start 2P

## How to Play
Simply open `index.html` in any modern web browser.

## GitHub Pages Deployment
1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Select the `main` branch and the root directory.
4. Your game will be live at `https://<your-username>.github.io/<repository-name>/`.
